# Vehicle Fleet Management System

## Automated testing 
Simulates keyboard strokes so no need for manual inputting.
1. In intellij search for` Edit >> Configurations Run`
2. Click `+` then select Create application testing
3. Under Build and Run Select the project's `main` 
4. click `modify` then select `redirect input` 
5. select `input.txt` from the project folder
6. happy testing!
